


### Основные команды для запуска проекта

- `npm install` — установка зависимостей,
- `npm run dev` — запуск версии для разработчика (PORT: 3000),
- `npm run build` — сборка prod версии.
- `npm run server` — запуск сервера на 3000 порте.
